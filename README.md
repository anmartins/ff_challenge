

# Read Me

### Setup

If ruby does not exist on your machine.
```
curl -sSL https://get.rvm.io | bash

rvm install 3.3.0
```

To kickstart the exercise

```
bundle install
```

To run the exercise

```
bundle exec ruby main.rb '4,3,6,2,1,7'
```
````
bundle exec ruby main.rb '4,3,6,2,1,7' 3
```

To run test exercise

```
bundle exec rspec
```
